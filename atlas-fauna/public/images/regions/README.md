# Region Photos

Папки для фотографий регионов:

- `public/images/regions/khabarovsk`
- `public/images/regions/primorye`
- `public/images/regions/amur`
- `public/images/regions/yevrey`
- `public/images/regions/sakhalin`

Рекомендация по именам файлов:

- `cover.jpg` - основная фотография региона
- `extra-1.jpg`, `extra-2.jpg` - дополнительные фото (на будущее)

Соответствие коду регионов:

- `Khabarovsk` -> `khabarovsk`
- `Primor'ye` -> `primorye`
- `Amur` -> `amur`
- `Yevrey` -> `yevrey`
- `Sakhalin` -> `sakhalin`
